# PDF Splitter

스캔한 PDF 한 파일을 **개인별 PDF로 나누고, 명단대로 이름을 붙여** 내려받는 도구.

여러 사람 분량이 이어 스캔된 서류(답안지, 설문지, 제출물 등)를 1인당 N쪽씩 잘라
`프로젝트명_학급_01_정화선.pdf` 형태로 저장합니다.

## 왜 만들었나

일반 PDF 분할 도구는 `문서_1.pdf ~ 문서_23.pdf`로만 뱉습니다.
23개를 명단 보며 손으로 이름 바꾸는 일이 진짜 노동입니다. 그 부분을 없앴습니다.

## 기능

- **1인당 쪽수 지정** (1~4쪽) 후 일괄 분할
- **이름 목록 붙여넣기** → 파일명 자동 부여 (엑셀에서 번호·이름 열 복사 가능)
- **썸네일 확인** — 조각마다 첫 쪽을 보고 사람이 맞는지 눈으로 검증
- **경계 조정** — 빈 쪽 삽입(밀기) / 첫 쪽 제거(당기기)로 어긋난 지점 교정
- **회전** — 전체 90°씩 순환, 조각별 개별 회전도 가능 (무손실)
- **제외 토글** — 미제출자 빈 종이를 결과에서 배제
- **개별 저장 / 전체 ZIP** 다운로드
- **자동 경고** — 총 쪽수가 나누어떨어지지 않거나 명단 수와 조각 수가 다르면 안내

## 개인정보

**모든 처리가 브라우저 안에서만 이루어집니다.** 파일이 서버로 전송되지 않고,
어디에도 저장되지 않으며, 새로고침하면 전부 사라집니다.
(대부분의 온라인 PDF 도구는 파일을 자사 서버로 업로드해 처리합니다.)

## 사용법

1. PDF 선택 (또는 끌어다 놓기), 1인당 쪽수와 파일명 접두어 지정
2. 이름 목록 붙여넣기 → `분할하기`
3. 썸네일로 조각 확인 — 어긋난 곳은 경계 조정, 빈 종이는 `제외`, 방향은 회전
4. 개별 저장 또는 `전체 ZIP으로 내려받기`
5. /* 5단계(지시문)는 ?plus=1 이 붙은 주소에서만 나타난다.
   가을 공개 시 PLUS 상수와 아래 두 곳의 조건만 지우면 모두에게 열린다. */

### 스캔 요령

**미제출자 자리에는 빈 종이를 넣어 스캔하세요.** 그래야 "N번 = (2N−1)~2N쪽"이
끝까지 어긋나지 않습니다. 빠뜨렸더라도 3단계의 경계 조정으로 교정할 수 있습니다.

## 이름 목록 형식

한 줄에 한 명, 파일에 담긴 순서대로. 아래 형식을 모두 인식합니다.

```
1	정화선          ← 번호 + 탭/공백/점/괄호 + 이름
2. 조은영
3) 장인석
안수빈             ← 이름만 (순서대로 번호 부여)
```

비워두면 순번만 붙습니다.

## 기술

단일 HTML 파일. 빌드 과정 없음. 정적 호스팅으로 충분합니다.

- [pdf.js](https://mozilla.github.io/pdf.js/) — 썸네일 렌더링
- [pdf-lib](https://pdf-lib.js.org/) — 분할·회전
- [JSZip](https://stuk.github.io/jszip/) — 일괄 다운로드
